If a new speech file is wanted, enter the filename in the input text box and speak the new speech after you submit the form. If you want to observe or delete a current speech file, enter the file from the select list. Choose time or spectrogram plots to observe and enter the frequency domain parameters for spectrogram plot observation.

The filename should be the same as the new speech you are creating. It should end with the wav extension. The Audio Wav Directory must be specified for play/delete operations.





Frequency Domain Parameters

Enter frequency domain configuration. FFTs are averaged with 50% overlap. Zero-padding is used as necessary given the number of segments to average and the FFT size. The data is windowed to reduce sidelobe spectral leakage with a cost of main lobe widening and reduced resolution. Words Only checkbox skips audio samples that don't contain words. Word window checks for words boundaries when Words Only is used.
